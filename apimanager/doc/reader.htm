reader/rlist 获取订阅列表 用户ID：uid(int)
订阅ID：rid(int)
rid=0为一级订阅列表		 {
result:[
{
"rid": 1,
"rname":"放开那三国",

        },

],

}
reader/mark 用户新增订阅 用户ID：uid(int)
订阅ID：rid(int)		 {
    "errorCode": 0
}
reader/del 用户取消订阅 用户ID：uid(int)
订阅ID：rid(int)		 {
    "errorCode": 0
}
reader/search 用户搜索订阅 用户ID：uid(int)
